Local Single-User Server

{% highlight r %} install.packages("opencpu", repos = c('http://rforge.net', 'http://cran.rstudio.org')) library(opencpu) {% endhighlight %}

Ubuntu Cloud Server

{% highlight bash %} sudo add-apt-repository ppa:opencpu/opencpu-1.0 sudo apt-get update sudo apt-get install opencpu {% endhighlight %}

PDF Server Manual

The OpenCPU server manual PDF is the primary reference for installing and managing OpenCPU servers.

Installing OpenCPU Apps

{% highlight r %} library(devtools) install_github("gitstats", "opencpu") install_github("stocks", "opencpu") {% endhighlight %}
