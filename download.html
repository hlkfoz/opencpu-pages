Ubuntu 14.04 Cloud Server

{% highlight bash %} #requires ubuntu 14.04 (Trusty) sudo add-apt-repository ppa:opencpu/opencpu-1.4 sudo apt-get update sudo apt-get install opencpu {% endhighlight %}

Suse / Fedora / CentOS Cloud Server

See this readme for downloading or building rpm packages.

{% highlight r %} sudo yum install opencpu {% endhighlight %}

Server Manual PDF

The OpenCPU server manual PDF is the primary reference for installing and managing OpenCPU servers.

Local Single-User Server

{% highlight r %} install.packages("opencpu") library(opencpu) {% endhighlight %}

Installing OpenCPU Apps

{% highlight r %} #install the "gitstats" app from github library(devtools) install_github("gitstats", "opencpu") #open in single-user server library(opencpu) opencpu$browse("/library/gitstats/www") {% endhighlight %}
