About the Library

The opencpu.js JavaScript client library builds on jQuery and provides Javascript interfaces to call R from within a web page via the OpenCPU API. The library works on all modern browsers, and is a solid foundation for building R web applications.

OpenCPU Apps

An OpenCPU app is an R packages which includes web page(s) that calls the R functions in the package over the OpenCPU API. This makes a convenient way to develop, package and ship portable, standalone R web applications.

{% highlight js %} //calls the function "rnorm" in the current package var jqxhr = opencpu.r_fun_call("rnorm", {n:10, mean:5}, function(session) { alert("Success! " + session.getLoc()); }).fail(function(){ alert("Failure: " + jqxhr.responseText); }); {% endhighlight %}
