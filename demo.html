× A public demo server is provided for anyone to get started with OpenCPU, and play around with apps and packages. It has a standard OpenCPU cloud server installation with approx 1000 popular CRAN packages in the /ocpu/library/. In addition you can try the /ocpu/cran API to install other CRAN packages.
× The server is mostly here to demo the API. Performance will vary (depending on how busy it is) and it might go down or be upgraded without warnings. To host important OpenCPU apps, please install your own OpenCPU cloud server.

The easiest way to share your scripts, documents, packages or apps on OpenCPU is by putting them on github/gist:
