opencpu.js

The opencpu.js javascript library provides some wrapper functions to call the API from Javascript. It is a nice foundation for developing OpenCPU apps.

Source code, documentation and examples are available at the Github repository.

appdemo

To get started, have a look at the OpenCPU app called appdemo. It contains some minimal examples demonstrating how to use the opencpu.js library to call R functions from Javascript using OpenCPU. Also see this blog post.

Function Call

{% highlight js %} //calls the function "rnorm" in the current package var jqxhr = opencpu.r_fun_call("rnorm", {n:10, mean:5}, function(session) { alert("Success! " + session.getLoc()); }).fail(function(){ alert("Failure: " + jqxhr.responseText); }); {% endhighlight %}

Draw Plot

{% highlight js %} //calls the function "plot" and displays the result in #mydiv var jqxhr = $("#mydiv").r_fun_plot("plot", { x : [3, 5, 3, 2, 4 ] }) {% endhighlight %}

JSON RPC

{% highlight js %} var jqxhr = opencpu.r_fun_json("rnorm", {n:10, mean:5}, function(data){ dosomething(data); }); {% endhighlight %}
